# Data Analytics Dataset

## 📌 Overview

This repository contains the dataset used for Data Analytics tasks and practice.

The dataset is provided in Excel format and can be used for data cleaning, analysis, visualization, and reporting.

## 📂 Repository Contents

- `Dataset for Data Analytics.xlsx` – Main dataset for analysis.

## 🎯 Objectives

The dataset can be used to:

- Perform data cleaning
- Analyze trends and patterns
- Generate insights
- Create charts and dashboards
- Practice Excel data analysis techniques

## 🛠️ Tools

- Microsoft Excel
- Google Sheets (optional)

## 📊 Skills Practiced

- Data Cleaning
- Sorting & Filtering
- Pivot Tables
- Charts
- Basic Data Analysis

## 🚀 How to Use

1. Clone this repository.
2. Download the Excel file.
3. Open it in Microsoft Excel.
4. Perform the required analysis.
